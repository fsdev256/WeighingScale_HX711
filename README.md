# WeighingScale_HX711
DIY weighing scale with HX711 ADC module and 1KG load cell bar

## Hardware
- ESP-01s
- HX711 ADC
- Load cell bar 1KG
- 4x 1.5V Alkaline Battery
- LM1117 Voltage Regulator
- 2x Capacitor (10 uF and 35uF)
